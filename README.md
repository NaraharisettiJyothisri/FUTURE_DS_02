# Customer Churn & Retention Analysis Dashboard

This project focuses on analyzing customer churn and retention patterns using Power BI. The goal of the analysis is to understand why customers leave, identify customer segments with higher churn rates, and discover factors that improve customer retention.

## Objectives

* Analyze customer churn trends
* Identify high-risk customer segments
* Study customer tenure and retention patterns
* Understand the impact of contract type and monthly charges on churn
* Generate actionable business insights

## ⛏️Tools Used

* Power BI
* Data Visualization
* Business Analytics

## 🔍Dashboard Features

* Churn Distribution Analysis
* Contract Type vs Churn Analysis
* Monthly Charges vs Churn Analysis
* Customer Tenure Analysis
* Interactive Visualizations and Insights

## Key Insights

* Customers with month-to-month contracts show higher churn rates.
* Higher monthly charges are associated with increased churn.
* Long-term customers are more likely to stay active.
* Contract type significantly affects customer retention.

## Business Recommendations

* Encourage long-term subscription plans.
* Provide retention offers for high-risk customers.
* Improve customer engagement strategies.
* Reduce churn through loyalty programs and personalized support.

This project demonstrates practical skills in customer retention analytics, churn analysis, dashboard design, and business-focused data storytelling using Power BI.
